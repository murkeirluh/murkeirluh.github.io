  1. A Pill A Day

  • Web application developer
  • HTML, CSS, Python/Django
  • May 2018
  • apilladay.herokuapp.com

A web app that allows doctors to create and update prescriptions and intake schedules for their patient/s. This web app is integrated with our smart medicine box and its accompanying mobile app that alerts the patient when it is time to take their medicine for the time of day. This was our capstone project for Computer Networks class (CS 145).

The web app was developed using Python/Django, with PostgreSQL for the database. Semantic UI was used for frontend. The app was deployed using Heroku. The project name is a play on "apple a day".

  1. Medical Center Manager

  • Front end developer
  • HTML, CSS, Python/Django
  • November 2017 - December 2017
  • Medical-Center-Manager on Github

This system ties together information from data.gov.ph HIV treatment centers, social hygiene clinic, and HIV testing hubs. The system helps give people information about an institution through ratings and feedback, as well as allow them to set appointments. Institutions can also manage these appointments and other information about services and products that they offer. This was our project for Database Systems class (CS 165).

I worked mostly on the frontend design of the web app, using HTML and CSS, and some Python (Django). Semantic UI was used for the app's interface, and MySQL for the database.

It hasn't been maintained after that semester, so the screenshots shown are only some that were shared among us groupmates (and, thus, are blurry/small)

  1. Libra v.2.0

  • Project Head, Scrum Master, and Lead Developer
  • Python
  • October 2016 - May 2017

A quiz bee software developed in Python, for UP Mathematics Club.

Libra is UP Math Club's software that is used in math competition events. It helps keep track of contestants who have pressed the buzzer, and it also functions as a timer for every question/round.

As a frequent Verdict (or Libra operator/handler) during our events, I thought that it was time to give the software an upgrade—I proposed Libra v. 2.0 to be our product for Software Engineering class (CS 191/192). It was completed in time to be officially deployed on UP Math Club's 44th Annual Nationwide Search for the Math Wizard, held 18 February 2017, ~3 months earlier than the deadline for our class.

This was developed with the help of my three awesome groupmates, and also with insights provided by a number of UP Math Club members. Our group won the Customer First Award in class because of our aggressive schedule to complete the app in time for Math Club's biggest event of the year.
